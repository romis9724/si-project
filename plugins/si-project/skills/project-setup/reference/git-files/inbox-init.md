# Project Inbox — 원격 작업 큐

> AI agent 비대기 상태에서 사용자가 모바일·웹·다른 디바이스에서 큐잉하는 "다음 할 일" 목록.
> `/si-project:project-inbox`가 본 파일 + GitHub Issues(label:si-project-inbox)를 합쳐 표시한다.
>
> **사용법**:
> - 어디서든 본 파일에 항목 추가 후 git push (GitHub 모바일 앱에서 직접 편집 가능)
> - 형식: 우선순위(P0/P1/P2) 헤더 아래 `- [ ] 항목` 1줄
> - 처리 완료 시 `- [ ]` → `- [x]` 마킹 또는 줄 삭제
>
> **입력 채널 대안**:
> - GitHub Issues에 `si-project-inbox` 라벨 부여 — 모바일 GitHub 앱·웹에서 즉시 추가
> - 폐쇄망 환경에서는 본 파일만 사용
