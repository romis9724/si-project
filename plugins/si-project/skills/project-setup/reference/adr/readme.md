# Architecture Decision Records (ADR)

본 디렉토리는 본 프로젝트의 주요 아키텍처 결정을 영구 기록한다 (MADR 형식).

## 새 ADR 작성
`/si-project:project-adr "결정 제목"` 호출로 다음 번호의 ADR을 자동 생성.

## 형식 (MADR 0.x)
- Status: Proposed / Accepted / Deprecated / Superseded by ADR-XXXX
- Context: 결정이 필요한 배경·제약
- Decision: 채택한 안
- Consequences: 긍정·부정 영향
- (선택) Alternatives Considered: 검토 후 기각한 안

## 인덱스

| 번호 | 제목 | 상태 | 일자 |
|------|------|------|------|
| 0001 | 스택 선택 (init 자동 생성) | Accepted | {DATE} |
